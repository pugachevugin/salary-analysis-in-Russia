# Анализ реальных зарплат в РФ (2000-2023)

Приложение на Streamlit для визуализации динамики зарплат с учетом инфляции.

## Функционал
* Загрузка данных Росстата из Excel.
* Сравнение номинальных и реальных зарплат по отраслям.
* Интерактивные графики с использованием Seaborn и Matplotlib.

## Как запустить локально
1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите приложение: `streamlit run app.py`
